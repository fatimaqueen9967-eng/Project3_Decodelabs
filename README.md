****🌐 Internship Portfolio Website
This is a responsive personal portfolio website developed as part of my
internship project using **HTML, CSS, and JavaScript**.
** Project Journey**

The initial version of this website was manually designed by me with a simple and basic layout.
Later, I improved and refined the design using modern UI/UX techniques and AI-assisted enhancements to make it more:
- Visually attractive  
- Interactive  
- User-friendly  
- Professionally structured  
This helped me better understand how real-world frontend projects are improved and optimized.
## 🚀 Features
- Modern hero section with animations  
- Login-based dashboard system  
- Fully responsive layout  
- Smooth scrolling and reveal animations  
- Animated counters  
- Interactive project cards (tilt effect)  
- Contact section with clean UI  
- Mobile-friendly navigation  
## 🛠️ Technologies Used
- HTML
- CSS 
- JavaScript (DOM manipulation, events)  
- Google Fonts
